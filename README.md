# ✊✋✌ Rock Paper Scissors Game

A simple **Rock Paper Scissors game** built using **HTML, CSS, and JavaScript**.
The user plays against the computer, and the score updates automatically after each round.

---

## 🎮 Features

* Interactive **Rock, Paper, Scissors gameplay**
* **Random computer choice**
* **Scoreboard** for both player and computer
* Displays **win, lose, or draw messages**
* Dynamic **message color change**
* Simple and responsive UI

---

## 🛠 Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript (DOM Manipulation)**

1. Clone the repository

```
https://github.com/syed-adnan-01/Stone-Paper-Scissor.git
```

2. Open the project folder

```
cd rock-paper-scissors
```

3. Open **index.html** in your browser.

---

## 🎯 How to Play

1. Click on **Rock**, **Paper**, or **Scissors**.
2. The computer will randomly select its move.
3. The winner is decided using the rules:

* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock

4. The **scoreboard updates automatically** after every round.
